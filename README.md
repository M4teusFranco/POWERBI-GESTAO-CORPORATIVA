### 📋 Descrição do Projeto

Este projeto foi desenvolvido em Power BI com foco na análise integrada de dados corporativos, contemplando Clientes, Funcionários e uma visão executiva consolidada (Resumo Geral).

O dashboard permite a visualização estratégica das informações organizacionais, facilitando análises financeiras, operacionais e comerciais por meio de gráficos interativos, indicadores e filtros dinâmicos.

Os dados foram estruturados a partir de bases em Excel, com modelagem relacional e uso de medidas DAX para construção de indicadores e análises comparativas.

#### ⚙️ Funcionalidades principais

Através de filtros e gráficos, é possível:

✅ Visualizar indicadores consolidados de clientes, funcionários e contratos  
✅ Analisar ticket médio e faturamento total  
✅ Comparar áreas e setores da organização  
✅ Avaliar distribuição geográfica de clientes e colaboradores  
✅ Identificar cliente com maior faturamento  
✅ Analisar massa salarial e estrutura organizacional  
✅ Filtrar dados por região, área, nível e período  
✅ Explorar dashboards integrados (Resumo Geral, Clientes e Funcionários)  

Este projeto utiliza dados estruturados em Excel integrados ao Power BI, proporcionando uma análise visual clara e dinâmica.

#### 🛠️ Ferramentas utilizadas

- Power BI Desktop  
- Base de Dados em Excel  
- Modelagem de Dados Relacional  
- DAX   
- Visualizações Interativas e Filtros Dinâmicos
